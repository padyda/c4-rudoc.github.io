#######################
Добро пожаловать в CodeIgniter4
#######################

CodeIgniter - это фреймворк для разработки приложений - инструмент - для тех, кто создает сайты с помощью PHP. Он позволяет ускорить разработку проектов по сравнению с написанием кода с нуля и предоставляет богатый набор возможностей для реализации рутинных задач, а такжей простой доступ к этим библиотекам. CodeIgniter позволяет вам сосредоточиться на своем проекте и минимизирует объем кода, необходимый для решения поставленной задачи.

CodeIgniter, насколько это возможно, остается гибким, позволяя работать так, как нужно вам, не принуждая к чему-либо. Ядро фреймворка имеет возможности простого расширения или, в случае необходимости, позволяет его полностью заменить. Вкратце, CodeIgniter - это пластичный фреймворк, который обеспечивает вас необходимым набором инструментов и при этом не мешает вести разработку.

*****************************
CodeIgniter вам подойдет?
*****************************

CodeIgniter вам подойдет, если:

-  Вы хотите фреймворк не требовательный к ресурсам системы.
-  Вам необходима отличная производительность.
-  Вам нужен фреймворк, почти не требующий настройки.
-  You want a framework that does not require you to use the command
   line.
-  You want a framework that does not require you to adhere to
   restrictive coding rules.
-  You are not interested in large-scale monolithic libraries like PEAR.
-  You do not want to be forced to learn a templating language (although
   a template parser is optionally available if you desire one).
-  You eschew complexity, favoring simple solutions.
-  You need clear, thorough documentation.
